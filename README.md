# Shaurya - Creative Portfolio Website

A premium, high-impact, minimalist orange-themed creative portfolio website built using standard semantic HTML5 and Tailwind CSS. The design system focuses on high-impact personal branding, combining rigorous modern minimalism with a warm gallery aesthetic, tactile interaction feedback, and bold typography.

## 🌟 Key Features

*   **Premium Modern Design:** A cohesive design system combining deep charcoal neutrals (`#1b1b1c`), a warm off-white surface (`#fcf8f9`), and vibrant orange accent colors (`#ab3600`).
*   **Fully Interconnected Pages:**
    *   **Home (`index.html`):** Introducing the identity, services grid (Bento layout), and selected works showcase.
    *   **About (`about.html`):** Professional studio profile, core design principles, and custom quote/mission block.
    *   **Experience (`experience.html`):** Beautifully structured modern timeline highlighting career achievements and skill badges.
    *   **Projects (`projects.html`):** Curated works gallery categorized by year and discipline.
    *   **Contact (`contact.html`):** Beautifully structured split layout containing an interactive input form and detailed connection card.
*   **Highly Responsive:** Tailored layouts for desktop, tablet, and mobile browsers using fluid grid alignments and responsive font weights.
*   **Micro-interactions:** Hover transitions, smooth scaling, custom active-states, and custom button redirections for cohesive web navigation.
*   **SEO Optimized:** Unique metadata titles and structured headers configured across all individual pages.

## 🛠️ Built With

*   **HTML5** (Semantic elements)
*   **CSS3** (Tailwind CSS via CDN for rapid loading and plug-and-play development)
*   **Google Fonts** (Inter typeface and Material Symbols Outlined icons)

## 🚀 Live Preview & Deployment

Simply clone this repository and open `index.html` in your web browser:

```bash
git clone https://github.com/harsharyann/Portfolio-Shaurya.git
cd Portfolio-Shaurya
# Open index.html in your default browser
```
